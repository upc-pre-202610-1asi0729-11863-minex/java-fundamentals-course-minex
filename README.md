# Java Fundamentals Course

Curso de responsabilidad social de Minex

## Configuración del repositorio

Este repositorio contiene los materiales del curso **Fundamentos de Java para Principiantes**.

### Estructura de carpetas

```
java-fundamentals-course-minex/
├── starter-files/       # Plantillas de código con las que los estudiantes comienzan cada lección
├── completed-examples/  # Código completo de referencia para cada lección
└── scripts/             # Guiones de grabación de cada lección en video
```

### ¿Cómo usar este repositorio?

1. **Clona el repositorio**
   ```bash
   git clone https://github.com/upc-pre-202610-1asi0729-11863-minex/java-fundamentals-course-minex.git
   ```

2. **Navega a la carpeta de la lección que te interesa**
   ```bash
   cd starter-files/        # para ver el punto de partida
   cd completed-examples/   # para ver la solución completa
   ```

3. **No necesitas instalar Java.** Todos los ejercicios se ejecutan en el navegador con [JDoodle](https://www.jdoodle.com). Copia el código del archivo correspondiente y pégalo en JDoodle.
